# Simple Web Notepad
A clean, modern web-based text editor that runs in your browser.

## What is it?
Simple Web Notepad is a lightweight text editor that works entirely in your web browser. It's perfect for taking quick notes, writing drafts, or editing text files without installing any software.

How to Use
- Open index.html in any modern browser
- Type your text in the editor area
- Save your work using the Save button or Ctrl+S
- Customize the look in Settings

## Controls
- Button	Shortcut	Action
- New	Ctrl+N	Clear the editor
- Open	Ctrl+O	Load a text file
- Save	Ctrl+S	Download as notes.txt
- Settings	-	Change appearance
- Help	-	View this guide
  
## Settings
You can customize:
- Fonts (choose from Google Fonts collection)
- Font size and line spacing
- Light or dark theme
- Background and text colors
- Text wrapping

## Privacy Notice
Everything stays in your browser. No data is sent to any server (except when loading fonts from Google Fonts).

## Browser Support
Works best in:
- Chrome / Edge
- Firefox
- Safari
- Opera
